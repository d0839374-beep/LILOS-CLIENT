# Lilos Visuals - Modern Forge 1.16.5 Mod

A modern visual mod for Minecraft Forge 1.16.5 featuring:

## Features

- **Custom ClickGUI** - Press Right Shift to open a modern, sleek ClickGUI with module toggles
- **HUD Renderer** - Displays FPS, coordinates, ping, speed, and armor status
- **Custom Main Menu** - Replaces vanilla main menu with a modern design
- **Custom Pause Screen** - Replaces vanilla pause screen with styled interface
- **Built-in Rendering Library** - Custom rendering utilities for gradients, rounded rectangles, and shadows

## Installation

1. Install Minecraft Forge 1.16.5 (version 36.2.39 recommended)
2. Download the mod JAR file
3. Place the JAR in your `.minecraft/mods` folder
4. Launch Minecraft with Forge profile

## Controls

- **Right Shift** - Open/close ClickGUI

## Modules (ClickGUI)

- ESP
- Fullbright
- Tracers
- ArmorHUD
- Coordinates
- FPS
- Ping
- Speedometer

## Configuration

Edit `config/lilosvisuals-client.toml` to customize:
- Enable/disable HUD
- Enable/disable custom menus

## Building from Source

```bash
./gradlew build
```

The compiled JAR will be in `build/libs/`
